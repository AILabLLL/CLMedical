# Mammoth Class-IL Result Converter

This project provides a **Python script** to convert **Class-IL results generated by the Mammoth framework** into a more **readable and easy-to-understand format (`resultYZ`)**.

The script is designed to be simple to use and requires **no command-line arguments**.
